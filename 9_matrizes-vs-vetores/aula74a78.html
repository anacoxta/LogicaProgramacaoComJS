Lógica de Programação com JS

Terminologia

VETORES = Array uni-dimensional (um nível)
ex: nomes = ['Maria', 'João', 'José', 'Ana']
nomes[2] === 'José' //true

MATRIZES = Array multi-dimensional (2+ níveis)
numeros = [[1,2,3], [4,5,6], [7,8,9]]
numeros[1] //[4,5,6]
numeros[1][2] //6
numeros[2][1] //8

Desafio

Em cada bimestre um aluno possui 4 notas de 0 a 10.
Dado o array notas = [[5,6,4,3],[2,8,5,6],[3,6,4,8],[2,8,4,9]]
mostrar na tela:

  • As notas,
  • A média de cada bimestre,
  • A média anual
